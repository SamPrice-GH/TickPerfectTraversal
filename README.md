# TickPerfectTraversal

A backtesting application with web UI for trading stratgies. Utilises intra-bar tick playback on order fill for maximum accuracy. Connect common data providers or donate your own raw collected tick data for others to access. Write custom strategies in Python and export them in compatible format for your trading platform (NinjaTrader, TradingView, MetaTrader etc.). 

This is a pet project that is only touched when I have free time.
